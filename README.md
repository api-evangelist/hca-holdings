# HCA Holdings (hca-holdings)

HCA Holdings is an alias for HCA Healthcare, one of the largest for-profit operators of healthcare facilities in the world, operating hospitals, surgery centers, freestanding emergency rooms, and urgent care centers. The legal entity rebranded to HCA Healthcare; this index tracks the legacy name. No public developer portal or production APIs are advertised.
